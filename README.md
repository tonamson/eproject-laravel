## Cài đặt CURL để dùng POST | GET
- Tải cacert.pem về: https://curl.haxx.se/docs/caextract.html
- Vào file php.ini tìm dòng **curl.cainfo** thay thế bằng curl.cainfo = path_file_pem
- Reset lại **xampp**
